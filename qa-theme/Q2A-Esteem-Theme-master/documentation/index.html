Installation

Installing theme & Plugin

  1. Extract esteem-theme.zip.

  2. upload Content of esteem-theme directory to qa-themes/ directory in your Q2A site

  3. Log in as Administrator, then Go to Admin > General and select 'Esteem-Theme' as your default theme.

  4. Go to Admin > Theme Options. there you can customize your theme.

if you had any problems installing or using this theme feel free to ask for help on Our Forum.

if you are paid customer you can contact me directly at QA-Themes.com

Making your site like our demo

visit "Admin > Users" and in "Default avatar" section upload an avatar and check it's option button. then in "Avatar size on question lists" field enter '20' and save the form. also activate "Show excerpt in question lists" and set it's value to something like 300.

Esteem Theme offers a wide range of customizations that you can easily apply on your theme.

here is a list of options that is available in your Theme options. to access this options visit Admin > Theme Options

Theme Options

  1. Excerpt in question list: this options let you to add a short description of your the question in lists.
  2. Background Customization: This options let you choose 'background color' & 'background image'. you can choose up to 350 backgrounds!
  3. Typography: you can change the Font & Font Size of nearly everything in this theme.
  4. CSS Style Files: you can choose any of CSS compressions

Add More Backgrounds

you have more backgrounds that you want to try on your theme? simply save them as .png files and upload the file in 'qa-themes/esteem-theme/patterns' and it will be added to backgrounds list.

Custom CSS styling

you can easily add more CSS styles to your theme or customize them. your theme might also come with a few custom CSS files for adding different colors, typography, etc ...

to add a new CSS file you can upload it in 'qa-themes/esteem-theme/styles' or you can edit one of pre-made styles.

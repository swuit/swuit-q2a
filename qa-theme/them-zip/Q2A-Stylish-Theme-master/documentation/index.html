Installing theme

Theme Installation Guide

  1. Download Theme.

  2. Extract stylish-theme directory from Stylish-Theme.zip.

  3. upload stylish-theme directory to qa-themes/ directory in your Q2A site

  4. Log in as Administrator, then Go to Admin > General and select 'stylish-theme' as your default theme.

  5. Go to Admin > Theme Options. there you can customize your theme.

Stylish Theme offers a wide range of customizations that you can easily apply on your theme.

here is a list of options that is available in your Theme options. to access this options visit Admin > Theme Options

Theme Options

  1. Excerpt in question list: this options let you to add a short description of your the question in lists.
  2. Background Customization: This options let you choose 'background color' & 'background image'. you can even add more backgrounds to the theme.
  3. Typography: you can change the Font & Font Size of nearly everything in this theme.
  4. CSS Style Files: you can choose any of CSS compressions types
  5. Custom CSS File: you can build a custom css file and add it to your theme and choose it in options page to add it to the theme.

Add More Backgrounds

you have more backgrounds that you want to try on your theme? simply save them as .png files and upload the file in 'qa-plugins/stylish-plugin/patterns' and it will be added to backgrounds list.

you can use Subtle Patterns to choose and download free backgrounds.

Custom CSS styling

you can easily add more CSS styles to your theme or customize them. your theme might also come with a few custom CSS files for adding diffrent colors, typography, etc ...

to add a new CSS file you can upload it in 'qa-plugins/stylish-plugin/styles' or you can edit one of pre-made styles.
